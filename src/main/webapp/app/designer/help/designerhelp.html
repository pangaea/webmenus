I. About Menu Designer

The Menu Designer is a tool for designing, maintaining and deploying restaurant menus. It's really simple to use and provides an intuitive interface. You'll be on your way in minutes.

With the Menu Designer you'll have the ability to preview changes before they are applied to the server, like a true development platform.

II. Basic Concepts

  1. Location

    2. The "Menu Designer" tool manages menus based on location. A "Location" describes a physical restaurant location or (in it's simplest form) a collection of "Menus".

    Location / Menu Hierarchy 
    Location
   |
   +-- Menu(s)
        |
        +-- Category(s)
               |
               +-- Item(s)
                    |
                    +-- Option(s)
                    |
                    +-- Portion(s) - additional


    Back to table of content

  2. Menus

    3. A "Menu" represents a basic restaurant menu which contains food or beverage choices (i.e. Lunch, Dinner, Dessert, etc...). A "Menu" can be turned on or off on a schedule (available via the main console).

    Name:The name of the menu used for display
    Hidden:Check this option to hide the menu from public view

    Back to table of content

  3. Categories

    4. A "Category" is used to describe a sub-section of a "Menu" (i.e. Appetizers, Salads, etc...).

    Name:The name of the category used for display
    Hidden:Check this option to hide the category from public view

    Back to table of content

  4. Items

    5. A menu "Item" is used to describe a food or beverage item (i.e. Grilled Cheese, Chicken Soup, etc...).

    Name:The name of the menu used for display
    Description:Detailed description of item
    Hidden:Check this option to hide the menu from public view
    Image:Image, from "Image Library"2, associated with the item
    Portions:Portions associated with the item

    Back to table of content

  5. Options

    6. An "Option" describes a choice given to the patrol when ordering an item (i.e. Hot Sauce, Extra Cheese, etc...). There are different types of options which can be used and each can be associated with specific price.

    Name:The name of the menu used for display
    Type:Select: Choose any option, Select-One: Choose single option
    Price:Price associated with each selected option
    Hidden:Check this option to hide the menu from public view
    Data:Selections associated with this option. To create multiple selections enter them one per line

    Back to table of content

  6. Portions

    7. A "Portion" is a term used to describe the different sizes available for a selected item (Small, Large, etc...). There is a single portion automatically associated with every item but additional portions can be associated with a single item, essentially creating many versions of that item.

II. Step-by-Step Instructions

  1. Creating a Menu

    2. Create a new menu by clicking the "Create Menu" button on the toolbar1. This will put a menu under the "Menus" node called "New Menu". By clicking on the new menu item you can change the name and other properties.


    Back to table of content

  2. Creating a Category

    3. Create a new category by first clicking on a "Menu". Then click the "Create New Category" button on the "Menu" properties page1. This will put a category under the selected "Menu" called "New Category". By clicking on the new category you can change the name and other properties.


    Back to table of content

  3. Creating a Item

    4. Create a new item by first clicking on a "Category". Then click the "Create New Item" button on the "Category" properties page1. This will put a item under the selected "Category" called "New Item". By clicking on the new item you can change the name and other properties.


    Back to table of content

  4. Creating a Option

    5. Create a new option by first clicking on an "Item". Then click the "Create New Option" button on the "Item" properties page1. This will put a option under the selected "Item" called "New Option". By clicking on the new option you can change the name and other properties.


    Back to table of content

  5. Creating an Additional Portion

    6. Create a new option by first clicking on an "Item". Then click the "Add Portion" button on the bottom of the "Item" properties page. This will add a new portion to the list. The order of the list can be changed by dragging and dropping. They can be deleted by clicking on the red "X".


    Back to table of content

  6. Deleting Objects

    7. To delete any of the above objects simply click on the object (i.e. Menu, Category, Item, or Option) and click the "Delete Selected" button in the toolbar1.


    Back to table of content

  7. Preview Menus

    8. To preview the menus with the designer you can click the "Preview All Menus" button in the toolbar. This will allow for viewing of the latest changes before saving them to the server.


    Back to table of content

  8. Saving Changes

    9. To save your changes to the server you can click the "Save Changes" button in the toolbar. This will commit all the changes made in the Menu Designer back to the server. IMPORTANT: If changes are made by one user while another user is working in the designer the changes may be overwritten.
